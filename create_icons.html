Cricket Extension Icon Generator

This page will help you create the required PNG icons for the Chrome extension.

Instructions:

  1. Open the SVG icon in a browser or image editor
  2. Export/save as PNG in the following sizes:
    • icon16.png (16x16 pixels)
    • icon48.png (48x48 pixels)
    • icon128.png (128x128 pixels)
  3. Save them in the icons/ directory

For quick testing, you can temporarily comment out the icons section in manifest.json:

//     "16": "icons/icon16.png",
